# Mission: Learn Go (first programming language)

## Why

Learn to code so I can build a real side project, automate repetitive tasks in my life or work, and finally understand how software and computers actually work — not just use apps blindly.

## Success looks like

- I can write and run small Go programs from scratch without hand-holding
- I understand what a program is, what "running code" means, and how files become output
- I can use Go to solve a small real problem (automation or a tiny tool) as a capstone

## Constraints

- Complete beginner — no prior programming experience; explain jargon before using it
- Intense pace: ~6+ hours per week
- Prefer solo learning with the agent as teacher; no communities for now
- Go was chosen partly because it was recommended as approachable for beginners

## Out of scope

- Job interview prep and leetcode grinding (for now)
- Deep computer science theory (compilers, operating systems internals)
- Web frameworks, databases, and deployment until fundamentals are solid
- Other languages (Python, JavaScript, etc.) unless needed for context
