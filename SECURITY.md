# Security Policy

## Supported Versions

| Version | Supported          |
|---------|-------------------|
| latest  | ✅                |

## Reporting a Vulnerability

If you discover a security vulnerability in go-learn, please open a GitHub
issue with the label "security" describing the issue. Do not open a public
issue for critical vulnerabilities — instead, contact the maintainers through
the repository's security advisory feature.

We will respond within 48 hours and work to resolve confirmed issues promptly.
