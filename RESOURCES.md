# Go Resources

## Knowledge

- [Tutorial: Get started with Go — go.dev](https://go.dev/doc/tutorial/getting-started)
  Official install + Hello World + `go run`. Use for: first real program, `go` command basics, modules intro. *Note: assumes some programming experience — pair with our lesson 1 for absolute beginners.*
- [A Tour of Go — go.dev/tour](https://go.dev/tour/)
  Interactive official intro to syntax, structs, interfaces, concurrency. Use for: after lesson 3–4, when basic mental models are in place.
- [Go by Example — gobyexample.com](https://gobyexample.com/)
  Annotated short programs for common tasks. Use for: quick lookup while practicing, not as a first read.
- [Effective Go — go.dev/doc/effective_go](https://go.dev/doc/effective_go)
  Idiomatic Go style guide. Use for: after you can write small programs — too early for week one.
- [How Computers Work (video) — Code.org, 5 min](https://www.youtube.com/watch?v=DKGZlaM_VU8)
  High-level mental model of input → processing → output. Use for: lesson 1 grounding. *Note: link may be unavailable in some regions — lesson 1 covers the same idea in text.*
- [Bits and Bytes explained — Khan Academy, 5 min](https://www.youtube.com/watch?v=bMmlCQPXJOU)
  Backup for lesson 1 mental model: how computers store and process information.
- [Install Go — go.dev/doc/install](https://go.dev/doc/install)
  Official install steps for macOS, Linux, Windows. Use for: verifying your toolchain.
